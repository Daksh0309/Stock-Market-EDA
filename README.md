# 📈 Stock Market Exploratory Data Analysis

## Overview
This project performs Exploratory Data Analysis (EDA) on real-time stock market data for Indian companies using Python. Data is fetched automatically using yfinance — no CSV needed!

## Features
- ✅ Live data fetching using yfinance API
- ✅ Stock price trend visualization
- ✅ 50-day and 200-day Moving Averages (TCS, Infosys, Reliance)
- ✅ Daily Returns analysis
- ✅ Correlation Heatmap between stocks

## Technologies Used
- Python, Pandas, Matplotlib, Seaborn, yfinance

## How to Run
Open the notebook in Google Colab and run all cells. Data is fetched automatically up to today's date!
